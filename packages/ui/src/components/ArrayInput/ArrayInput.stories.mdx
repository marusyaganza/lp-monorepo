# ArrayInput & InputWithButton

A versatile set of input field components supporting multiple states—**Default**, **Focus**, **Error**, and **Disabled**—with password visibility, definition lists, and example fields. Includes action buttons for adding/removing entries where relevant.

[View design in Figma](https://www.figma.com/design/uLznHs3pU0bQqy7Wwn3xQx/Design-system?node-id=1865-1490&t=gAq1U2uU13jxhVYo-1)

## Component Features

- **Field Types:** Text, Password, Definition, Example
- **States:** Default, Focus, Error, Disabled
- **Actions:** Add/Remove buttons for lists, password visibility toggle
- **Labels & Error Text:** Accessible labels, inline error messaging
