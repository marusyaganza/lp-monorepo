# Button

A versatile button component supporting five variants (Primary, Secondary, Tertiary, Success, Danger) with customizable icon, loading state, different sizes, action styles, and shapes (circle or rectangular).  
This component implements accessible HTML buttons and is suitable for general CTAs, forms, and toolbar actions.

## Component Variants, States & Styles

The design supports visual states for `Default`, `Hover`, `Focus`, and `Disabled` (see reference below):
